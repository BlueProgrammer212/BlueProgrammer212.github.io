What is Self-XSS (Cross-Scripting attack)?

Self-XSS is a type of attack used by hackers to gain access on the victim's account. The victim runs a malicious source code unto their web console, unknowningly revealing their personal information to the attacker.

How does it work?

Clients are connected in one particular server, the attackers will then make you run a malicious program to gain access to your account. HTTPS (HyperText Transfer Protocol Secured) cookies are used to save your information in the web, they are saved only in your device, so no one can access it. However, the attackers will trick you into running a malicious code to make you send the cookies in a different backend server.

Why is it so dangerous?

This is dangerous, because the cookie contains your personal information, the attacker can also get your IP (Internet Protocol) address by redirecting you into a phishing link programmatically.
